# Değişiklik Günlüğü

## [1.1.0] - 2025-02-22

### Eklenen
- **Temalar**: 6 tema (Koyu, Açık, Mavi, Yeşil, Turuncu, Pembe); Ayarlar > Özelleştirme
- **Özelleştirme**: Tema seçimi, Windows ile birlikte başlat seçeneği
- **Detaylı inceleme**: En yeni ölçüm üstte; pencere açıkken canlı güncelleme
- **Odak yardımı**: Uygulama içinden aç/kapa (checkbox)
- **Windows güç ayarları**: "Windows güç ayarlarını aç" artık Güç Seçenekleri (powercfg.cpl) açıyor
- **Bakım**: %TEMP% temizleme, RAM (çalışma kümesi) küçültme (sadece bu uygulama, güvenli)
- Windows 10/11 uyumluluğu; tema değişince arayüz anında güncellenir (DynamicResource)

### Düzeltilen
- Detay penceresi tablosunda "en yeni üstte" sıralama ve metin
- Güç ayarları düğmesi artık doğru pencereyi açıyor
